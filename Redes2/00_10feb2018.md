### Cable de cobre

* En el cable viaja una RF, señal analoga, es electricidad, tiene fuerza y dirección.
* Se trenza para dispersar los vectores de fuerza.
* XDSL => tecnologia para servir comunicaciones.
* CAT 1 - CAT 8A
  * Velocidad => b/s
  * Ancho de banda => Hz
  * TIA 568 A/B => configuración de colores
  * TX / RX => envio y recepción de datos
  * Relación categoria / precio

### Medios no guiados

* Frecuencias
* No usan cables

**Reflexión**  => Reflejo, retro reflexión => señales de tráfico
**Refracción** => separa la luz incidente (efecto pajilla rota)

TIA / EIA 606 => Rotulación y documentación
* Esquema de administración uniforme de cableado
  * Etiquetado de terminaciones, medios, rutas
  * Etiquetas auto laminadas
  * Rutas etiquetatas en puntos de terminación y 2 registros intermedios
  * Etiquetado en cable, en el exterior de las tapas en el área de trabajo
  * Color para terminaciones de voz es **azul**
  * Color para terminaciones de datos es **blanco**
  * Color voz/datos en cuanto de equipo es **negro**
